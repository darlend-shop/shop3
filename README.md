# 🛒 DARLEND — Магазин цифровых товаров PUBG Mobile & Brawl Stars

Готовое решение для продажи игровой валюты (UC, гемы, Brawl Pass) через веб-интерфейс Telegram WebApp (Mini App). Проект бесплатный в обслуживании, исходный код и сайт размещаются прямо на GitHub Pages.

---

## 🛠️ Функционал магазина

*   **Интерфейс DARLEND:** Современная темная тема, адаптированная под экраны мобильных телефонов.
*   **Гибкий заказ:** Автоматическая смена полей ввода: для PUBG запрашивается UID, для Brawl Stars — Email и код верификации Supercell.
*   **Оплата в 1 клик:** Покупатель может скопировать номер карты Ozon Банка простым нажатием на нее.
*   **Панель администратора:** Заявка с UID/Email и кодом мгновенно отправляется владельцу в Telegram с кнопками подтверждения.
*   **Система отзывов:** Возможность публикации и просмотра отзывов реальных клиентов.

## 🚀 Пошаговая инструкция по запуску Фронтенда (Сайта)

### Шаг 1. Создание репозитория
1. Зарегистрируйтесь или войдите в свой аккаунт на [GitHub](https://github.com).
2. В правом верхнем углу нажмите на иконку **`+`** и выберите **New repository** (Новый репозиторий).
3. В поле **Repository name** укажите название: `darlend`.
4. Сделайте репозиторий публичным (**Public**).
5. Нажмите кнопку **Create repository**.

### Шаг 2. Загрузка файлов
1. В созданном репозитории нажмите **uploading an existing file** (загрузить существующий файл) или создайте файлы вручную через кнопку **Create new file**.
2. Создайте файл `index.html` и вставьте в него код интерфейса магазина.
3. Создайте файл `README.md` и вставьте этот текст.
4. Нажмите **Commit changes** внизу страницы, чтобы сохранить файлы.

### Шаг 3. Включение бесплатного хостинга (GitHub Pages)
1. Внутри вашего репозитория перейдите во вкладку ⚙️ **Settings** (Настройки) на верхней панели.
2. В левом меню найдите раздел **Code and automation** и выберите **Pages**.
3. В блоке **Build and deployment** найдите параметр **Source**. Выберите значение **Deploy from a branch**.
4. В строке **Branch** вместо `None` выберите ветку `main` (или `master`), а в соседней папке оставьте `/ (root)`.
5. Нажмите кнопку **Save**.
6. Подождите 1–2 минуты. Обновите страницу. Сверху появится зеленая плашка с готовой ссылкой на ваш сайт (например, `github.io`).
